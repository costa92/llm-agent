<objective>
The RAG SDK records structural trace data (chunk IDs, routing, fusion) but
no cost or latency: no stage durations, no model-call counts. This slice
adds the measurement half of RAG-OBS-01 — a new `obs` package of metric
types, a context-scoped call counter, and instrumentation that records
per-stage wall-clock durations and embedding/generation call counts into the
public surface of every import, retrieve, and ask flow.

Token accounting is **17-02** — this slice defines `obs.TokenUsage` and the
`obs.Metrics.Tokens` field but leaves them zero-valued.
</objective>

<tasks>
1. Create `obs/obs.go` (package `obs`, imports `context`, `sync/atomic`,
   `time` only):
   ```go
   type StageTiming struct { Stage string; Duration time.Duration }
   type CallCounts  struct { Embed, Generate int }
   type TokenUsage  struct {
       PromptTokens, CompletionTokens, TotalTokens int
       Estimated bool
   }
   type Metrics struct {
       TotalDuration time.Duration
       Stages        []StageTiming
       Calls         CallCounts
       Tokens        TokenUsage
   }
   ```
   - `Counter` with `atomic.Int64` embed/generate tallies; `NewCounter()`,
     `AddEmbed(n int)`, `AddGenerate(n int)`, `Counts() CallCounts`.
   - `WithCounter(ctx, *Counter) context.Context` and
     `CounterFrom(ctx) *Counter` (returns nil when absent — never panics).
   - Doc-comment `TokenUsage` / `Metrics.Tokens` as "populated by token
     accounting (17-02)".
2. Create `rag/instrument.go`:
   - `countingEmbedder{inner embed.Embedder}` implementing `embed.Embedder`:
     `Embed` calls `obs.CounterFrom(ctx).AddEmbed(1)` (nil-safe) then
     `inner.Embed`; `Dimension` forwards.
   - `countingModel{inner generate.Model}` implementing `generate.Model`:
     `Generate` calls `obs.CounterFrom(ctx).AddGenerate(1)` (nil-safe) then
     `inner.Generate`.
3. Wire the decorators in `rag.New` (`rag/system.go`):
   - Wrap `emb` into a `countingEmbedder` **after** `emb.Dimension()` is read
     for the default store, and use the wrapped embedder for `s.embedder` and
     for the default `DenseRetriever.Embedder`.
   - Wrap `opts.Model` into a `countingModel` once; use the wrapped model for
     `s.model` and for the default `LLMExpansionPreprocessor.Model`.
   - A caller-supplied `opts.Retriever`/`opts.Preprocessor` is left as-is
     (documented limitation — see RESEARCH Decision 3).
4. Add `Metrics obs.Metrics` to: `rag.Diagnostics`, `retrieve.Trace`,
   `rag.ImportTrace`, and `ingest.ImportResult`. (`retrieve` and `ingest`
   gain an `obs` import — both are safe: `obs` is a leaf package.)
5. Instrument `rag/retrieve.go` `retrieve()`:
   - Resolve the counter: `c := obs.CounterFrom(ctx)`; if nil, create one and
     attach it to `ctx`. Snapshot `before := c.Counts()`.
   - Time `pre.Process` and `ret.Retrieve` as stages `preprocess` /
     `retrieve`; record `TotalDuration`.
   - `Calls` = `after - before` diff (correct whether standalone or nested).
   - Set `trace.Metrics` before firing `OnRetrieve` and returning.
6. Instrument `rag/ask.go` `Ask()`:
   - Create a fresh `obs.Counter`, attach to `ctx` (ask is always top-level).
   - Time stages `retrieve`, `rerank` (only when reranking runs), `pack`
     (only when packing runs), `generate`; record `TotalDuration`
     end-to-end and `Calls` from the counter.
   - Assign the assembled `obs.Metrics` to `Diagnostics.Metrics`.
7. Instrument `rag/import.go` `Import()`:
   - Time the split+embed loop (stage `embed`) and `store.Upsert` (stage
     `upsert`); record `TotalDuration`.
   - `Calls.Embed` = the existing `embedCount`; `Calls.Generate` = 0.
   - Set the `Metrics` on the returned `ImportResult` **and** on the
     `ImportTrace` passed to `OnImport`.
8. Tests `obs/obs_test.go`:
   - `Counter` tallies `AddEmbed`/`AddGenerate` and `Counts()` reflects them;
     `CounterFrom` on a bare context returns nil and `AddEmbed` on nil is a
     safe no-op.
9. Tests in `rag` (e.g. `rag/instrument_test.go`):
   - After an `Ask` with a scripted model + the default wiring,
     `answer.Diagnostics.Metrics` has `TotalDuration > 0`, a non-empty
     `Stages` with a `generate` entry, and `Calls.Generate >= 1`.
   - After an `Import`, `result.Metrics` has `Calls.Embed == result.Chunks`
     and `TotalDuration > 0`.
   - A retrieve records `retrieve`-stage timing in its trace `Metrics`.
   Use the existing scripted model + hash embedder; assert durations are
   `>= 0` and stage names are present (do not assert exact nanoseconds).
10. Run every command in `<verify>`.

Out of scope:
- Token counts — `obs.Metrics.Tokens` stays zero; that is 17-02.
- `otelrag` metric emission — that is 17-03.
- Counting calls inside caller-supplied retrievers/preprocessors.
</tasks>

<objective>
Add the opt-in eager `PrewarmCommunityReports`, surface community
attribution on local graph retrieval, and ship a deterministic global-search
worked example. Completes RAG-GRAPH3-04 and Phase 24.
</objective>

<context>
- `System.AskGlobal` + the lazy report cache come from 24-02;
  `graph.CommunitySummarizer` / `CommunityContentHash` and the
  `store.CommunityStore` report methods from 24-01.
- v0.7 `retrieve.GraphRetriever` (`retrieve/graph.go`) is entity-anchored
  local search; `GraphTrace{SeedEntityIDs, ReachedEntityIDs, MaxHop}`.
- `examples/graphrag_example_test.go` (v0.7) is the worked-example template
  — `DictionaryEntityExtractor` + in-memory store + scripted model, a stable
  `// Output:`.
- See `24-RESEARCH.md` Decision 4. Keystone KG3-2 (eager is a thin opt-in).
</context>

<tasks>
1. `rag/global.go` — add
   `func (s *System) PrewarmCommunityReports(ctx, namespace string) (int, error)`:
   type-assert `store.CommunityStore` (absent → `0, nil`); for every
   community, if no fresh cached report (missing or `ContentHash` mismatch),
   `Summarize` (require `s.communitySummarizer`) and `PutCommunityReport`.
   Return the count generated. Same summarizer, same cache as the lazy
   path — just called eagerly.
2. `retrieve/graph.go` — community attribution on local retrieval:
   `GraphTrace` gains `CommunityIDs []string` (additive). When the store is
   a `store.CommunityStore`, `GraphRetriever.Retrieve` looks up the
   communities and records the IDs of the communities its reached entities
   belong to (sorted, deduped). When the store is not a `CommunityStore`,
   the field is nil — no behavior change, graceful.
3. `examples/graphrag_global_example_test.go` — `Example_graphRAGGlobal`:
   a fully deterministic end-to-end wiring — `DictionaryEntityExtractor` +
   `LouvainDetector` in `Options`, a scripted `generate.Model` for the
   summarizer and the map/reduce steps, import a small fixed corpus, call
   `AskGlobal`, print the answer and the consulted community count — with a
   stable `// Output:`. No live model.
4. Extend `rag/global_test.go` — a `PrewarmCommunityReports` test: prewarm
   fills the cache (returns the community count); a subsequent `AskGlobal`
   makes no summarizer call (all cache hits); prewarm after a graph change
   regenerates only the stale reports.
5. Extend `retrieve/graph_test.go` — assert `GraphTrace.CommunityIDs` is
   populated when the store is a `CommunityStore` with communities, and nil
   otherwise.
6. Run every command in `<verify>`.

Out of scope:
- `docs/graphrag.md` Tier-3 update — Phase 25 (25-03).
- global-search eval harness — Phase 25 (25-02).
</tasks>
